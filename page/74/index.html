Magento javascript stopped working + solution

Aug 11, 2011 · 1 minute read
magento I’ve just spent to long trying to fix an infuriating problem with a simple fix. If you create a new theme from scratch and suddenly realise that none of your magento javascript is working, it could be because you’ve included jQuery and forgotten to include the noConflict argument. If you have included jQuery, then make sure the following line is included at the top of your page. jQuery.noConflict();
