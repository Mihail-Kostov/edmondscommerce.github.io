magento If you are struggling to understand why you are getting a popup on the Magento checkout with the simple message “undefined” then here is what you need to do. Use Fire Fox and install the extension Firebug Switch Firebug on and go to the console tab Hit the place order button and you will see the HTML response to the AJAX request (how did we cope with AJAX before Firebug!!)
