magento If you are running multi site Magento with different currencies and are finding that the sales reports on the stores that don’t use the Magento default currency all show zero this probably means the currency conversion rates have not been configured. Due to the way Magento orders work existing orders will not have there conversion rate updated. There is no way to do this from the admin, the only real option is to update the orders directly in the database.
